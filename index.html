IoT-Enabled Smart Conveyor System

Designed an IoT-powered conveyor system integrating quality control, inventory tracking, and automation using Arduino Uno, ESP32, and sensors.

My team and I developed an IoT-driven conveyor system that automates quality control and inventory management. The system features a load cell with HX711 amplifier for precise weight measurement, an SG-90 servo motor for automated load placement, and an ultrasonic sensor for real-time load detection and counting. If underweight items are detected, the conveyor halts via an L298N motor driver to ensure quality assurance. Using ESP32, the system transmits live data to the cloud for monitoring stock levels, counts, and overall system performance.

  • Accurate weight-based activation using Load Cell + HX711
  • Servo-controlled mechanical arm for consistent placement
  • Ultrasonic sensor with LED feedback for real-time counting
  • Fail-safe stoppage for underweight loads
  • Cloud-based monitoring with ESP32 IoT integration
